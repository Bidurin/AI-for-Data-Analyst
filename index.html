13 / 13
You Now Know This

Neural Networks — In a Nutshell

1
AI → ML → DL
The Hierarchy
DL is a subset of ML which is a subset of AI
2
NEURON
The Unit
Receives inputs, multiplies by weights, fires if above threshold
3
WEIGHTS
The Memory
Numbers on connections — everything the network learns lives here
4
ACTIVATION
The Gate
ReLU / Sigmoid / Softmax — injects non-linearity
5
LAYERS
The Stack
Input → hidden(s) → output. Each layer transforms features
6
DEPTH
The Power
More layers = more abstraction. Edges → shapes → objects
7
TRAINING
The Loop
Forward pass → loss → backprop → weight update × thousands
8
OVERFITTING
The Trap
Too complex = memorises data, fails on new inputs
The big idea: A neural network is just millions of weights, adjusted until the right numbers produce the right outputs.
